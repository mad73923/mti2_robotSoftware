# System config module for STM32L4

This module configures the System Clock, Flash access and Peripheral clocks.

| Property | Value     |
| -------- | --------- |
| System Clock Source | PLL (MSI) |
| SYSCLK[MHz] | 80     |
| HCLK[MHz] | 80       |
| AHB Prescaler | 1    |
| APB1 Prescaler | 1   |
| APB2 Prescaler | 1   |
| MSI Frequency[MHz] | 4 |
| PLL_M    | 1         |
| PLL_N    | 40        |
| PLL_R    | 2         |
| Flash Latency(WS) | 4 |
| SysTick Period | 1ms |
